Usermanagement-School-Activity

A simple **User Management System** school activity project with admin and user profiles.  
Built with **PHP** and **CSS**, this app lets users register, log in, and manage profiles — while an admin can view and edit users.

## Features

✔️ User registration & login  
✔️ Admin dashboard for managing users  
✔️ User profile page  
✔️ Password change & logout  
✔️ Simple and responsive UI

- **HTML/CSS** – Frontend layout & styling
